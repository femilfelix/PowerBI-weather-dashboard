# 🌤️ Weather Analysis Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Weather Analysis Dashboard** built using Power BI, designed to provide real time and forecast based insights across multiple cities. It consolidates key weather metrics into a clean and user friendly interface for effective data analysis and decision-making.

## 🚀 Key Features

* Real time and forecast weather insights
* Analysis of temperature, humidity, and rainfall trends
* Air quality and pollutant level monitoring
* Interactive filters for city wise exploration
* Clear visualization using charts and KPI cards

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query (Data Transformation)
* DAX (Data Analysis Expressions)
* Weather API

## 📊 Data Processing

* Extracted data from Weather API
* Performed data cleaning and transformation using Power Query
* Created calculated measures and KPIs using DAX
